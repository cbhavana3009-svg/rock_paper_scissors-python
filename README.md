# 🎮 Rock Paper Scissors (Python)

A simple command-line Rock Paper Scissors game built using Python.

## 🚀 Features
- User vs Computer gameplay
- Random choice generation
- Input validation
- Score tracking

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Install Python
2. Run the file:
   python game.py
